Welcome to TaxSYS. The first step to setting up TaxSYS is configuring the Tax and Value options.

In order for TaxSYS to properly Tax miners and record payments it needs to know which Corporations to Tax, and which Corporations to receive payments from.

Only Corporations that you or another Director are CEO of will show in this list. If you are missing a Corporation you need to have the CEO of that Corporation to log in with this link.

{{ request.scheme }}://{{ request.get_host }}{% url 'director_login' tenant.token %}

Warning: This URL only works once and any TEST Auth account that uses it setup with Director roles in TaxSYS.
